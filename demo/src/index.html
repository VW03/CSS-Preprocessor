VWCSS Test Pagina
Alle mogelijke features staan op deze pagina!
Variabelen
Dit element (<a>) gebruikt een variabele "main-color" om de tekstkleur te veranderen.
CSS selectors
  • De nav op deze pagina gebruikt een kind selector.
  • Elke titel op deze pagina gebruikt een klasse selector.
  • Er is een margin van 2 pixels rond heel de pagina door de alles selector.
  • Het <article> element voor de Nesting en Parent Selectors gebruikt een id selector.
  • De header en footer gebruiken een groep selector om dezelfde kleur te hebben.
  • Het element <a> op deze pagina gebruikt een element selector.
  • Het element <ul> op deze pagina gebruikt een attribuut selector.
  • Dit element <li> gebruikt dezelfde attribuut selector, maar met een specifieke waarde om meer padding links te hebben.
